Contribute to your language

We want to support every language that Wingspan was translated into, so truly anyone can freely look up any card. Currently many languages are missing translated names for some birds, translations of their powers or appendix info. There might be typos in the translation or some languages are not supported at all!

Please help! If you own the game in a language that is not fully translated or you've found a typo and want to contribute, follow this link or write me directly at matej.cief@gmail.com

Many thanks in advance!
